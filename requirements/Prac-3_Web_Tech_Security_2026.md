#### Task 1

By now you should have a website that is mostly made of HTML and CSS Code. In this practical, you will be creating a **.js and JS_index.html** files that will be used in adding the behaviour and style to your existing website. Make sure that these two files are in the same directory as the original HTML file. **You are not allowed to use any framework for this practical including but not limited to jQuery, Next JS, Svelte JS, Nuxt JS, Jasmine, Jest, Bootstrap, Node.js, Spring boot e.t.c. The use of any framework will lead to loss of 75% of your overall mark for this prac.**

You are expected to perform the following tasks (not arranged in any order) using Javascript and HTML (where necessary):

1. In your second week using CSS, you transformed your boring HTML menus into good-looking navigation bars. You also created mobile-friendly, responsive navigation bars. This week we will use Javascript to make changes to that. The JavaScript navigator object is used for browser detection. It can be used to get browser information such as appName, appCodeName, userAgent etc. There are many properties of navigator objects that return information to the browser. Use at least 5 Navigator Object Properties to make changes to your existing Website.

2. The navigator object is the window property, so it can be accessed by: `window.navigator` or `navigator`. JavaScript navigator objects have two methods, use both of them.

3. When an HTML document is loaded into a web browser, it becomes a document object. The document object is the root node of the HTML document. The document object is a property of the window object. Everything else revolves around the DOM. The HTML DOM has at least 50 properties and methods. Use at least 10 properties and methods to transform your existing HTML document. How you use them is up to you.

4. Using Javascript add behaviour and beautify your existing Slideshow Gallery.

5. Comments are a MUST have in your files for this week.

6. In your second week of this course, you created multi columns with CSS using CSS Multi-column Properties that allowed you to construct Multicolumn Layouts. Show me how you can make improvements on this using JavaScript. Add this to your `.js` file.

7. HTML DOM events allow JavaScript to register different event handlers on elements in an HTML document. Events are normally used in combination with functions, and the function will not be executed before the event occurs. In your `.js` file, create at least 10 functions to suit the current functionalities of your website. There are at least 90 HTML DOM Events, your website should have at least 10 of those. How you use them should match with your current website functionality.

8. Make sure that each input on all your forms is validated and that no wrong input is found. Form validation generally performs two functions: **Basic Validation** - which you have already done with HTML. Check your HTML code to ensure the forms must verify if fields are mandatory fields are filled in. It would require just a loop through each field in the form and check for data. With JavaScript you will be expected to do **Data Format Validation** i.e., the data that is entered must be checked for correct form and value. Your code must include appropriate logic to test correctness of data. This should be apparent in your `.js` file and website.
